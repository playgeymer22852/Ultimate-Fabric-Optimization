## 📊 Состава модов / Mod List

### ⚡ Увеличение FPS и Рендеринг / Performance & Graphics
*   **Sodium** — *[RU]* Радикально поднимает FPS и снижает нагрузку на видеокарту. / *[EN]* Radically boosts FPS and reduces GPU load.
*   **Sodium Extra** — *[RU]* Дополнительные настройки графики в меню Sodium (анимации, частицы). / *[EN]* Adds extra graphical settings to the Sodium menu.
*   **Reese's Sodium Options** — *[RU]* Делает меню настроек Sodium очень красивым и удобным. / *[EN]* Makes the Sodium options menu clean and searchable.
*   **ImmediatelyFast** — *[RU]* Оптимизирует рендеринг текста, шрифтов, табличек и полосок здоровья. / *[EN]* Optimizes rendering of text, signs, and entity HUDs.
*   **BadOptimizations** — *[RU]* Убирает фоновые лишние вычисления рендеринга неба и интерфейса. / *[EN]* Removes unneeded background calculations for sky and UI rendering.

### 🎯 Отсечение невидимого / Culling
*   **Entity Culling** — *[RU]* Пропускает рендеринг мобов и сундуков, которые находятся за стенами. / *[EN]* Skips rendering entities and chests hidden behind walls.
*   **More Culling** — *[RU]* Оптимизирует просчет листьев деревьев, травы и жидкостей. / *[EN]* Optimizes culling for leaves, grass, and fluids.

### 🧠 Логика, Движок и Память / Engine & RAM
*   **Lithium** — *[RU]* Полностью переписывает физику игры и ИИ мобов для плавности мира. / *[EN]* Overhauls game physics and mob AI for better server/client ticks.
*   **ModernFix** — *[RU]* Исправляет баги движка Minecraft и убирает утечки памяти. / *[EN]* Fixes bugs in the Minecraft engine and eliminates memory leaks.
*   **FerriteCore** — *[RU]* Кардинально сжимает оперативную память, уменьшая потребление RAM. / *[EN]* Dramatically reduces memory usage by compacting RAM data.
*   **Better Block Entities** — *[RU]* Оптимизирует рендеринг сложных блоков (сундуков, спавнеров). / *[EN]* Optimizes rendering for tile entities like chests and spawners.
*   **Too Many Entities** — *[RU]* Стабилизирует игру при огромном количестве предметов на экране. / *[EN]* Stabilizes the game when too many items or entities are present.

### 🗺️ Генерация мира и Сеть / World & Network
*   **C2ME** — *[RU]* Многопоточный движок для идеально плавного полета и прогрузки чанков. / *[EN]* Multi-threaded chunk engine for stutter-free world loading.
*   **Chunky** — *[RU]* Позволяет заранее сгенерировать карту мира для экономии ресурсов. / *[EN]* Pre-generates world chunks to prevent lag spikes during exploration.
*   **Dynamic FPS** — *[RU]* Снижает FPS до 15 кадров, когда игра свернута. / *[EN]* Lowers FPS to 15 when the game window is unfocused or minimized.
*   **Fast Server Pings / Fast IP Ping** — *[RU]* Оптимизирует сетевой код для снижения пинга. / *[EN]* Optimizes network code for faster server pings and lower latency.

### 🔊 Аудио-атмосфера и Интерфейс / Audio & UI
*   **Sound Physics Remastered** — *[RU]* Добавляет реалистичное физическое эхо и реверберацию звуков. / *[EN]* Adds realistic sound physics, echoes, and cave reverberation.
*   **AmbientSounds** — *[RU]* Наполняет мир Minecraft живыми звуками природы. / *[EN]* Fills the world with realistic ambient sounds (birds, wind, caves).
*   **Presence Footsteps** — *[RU]* Полностью перерабатывает и улучшает звуки шагов по блокам. / *[EN]* Completely overhauls footstep acoustics for every block type.
*   **Iris Shaders** — *[RU]* Самый быстрый и современный движок для поддержки шейдеров. / *[EN]* Modern and fast shader engine with OptiFine shader pack support.
*   **Mod Menu** — *[RU]* Добавляет в главное меню удобный список всех установленных модов. / *[EN]* Adds a clean mod listing screen to the main menu.
